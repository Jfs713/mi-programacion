# 🖥️ Comandos básicos de terminal

## 📂 Navegación
- `pwd` → ver ruta actual
- `ls` → listar archivos
- `cd carpeta` → entrar en carpeta
- `cd ..` → subir carpeta

## 📁 Crear cosas
- `mkdir nombre` → crear carpeta
- `touch archivo.c` → crear archivo

## ❌ Borrar
- `rm archivo.c`
- `rm -r carpeta`

## ⚙️ Compilar y ejecutar C
- `gcc programa.c -o programa.exe`
- `./programa.exe`

## 🔁 Git básico
- `git add .`
- `git commit -m "mensaje"`
- `git push`
